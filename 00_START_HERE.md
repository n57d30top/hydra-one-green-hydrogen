## What This Is

Hydra One is a bounded, advisory-only computational lane for green-hydrogen catalyst discovery. In this packet, the main deliverable is a reviewed set of four top catalyst candidates plus surrogate structure artifacts for external technical inspection.

## What This Is Not

- not a wet-lab result
- not a DFT-backed crystal-truth claim
- not a synthesis protocol
- not a commercialization or deployment readiness packet

## Main Question For Review

Given the stated limits, do the candidate ranking, surrogate structure assumptions, and provenance package form a credible computational basis for follow-on scientific discussion or for a stricter next-step modeling lane?
